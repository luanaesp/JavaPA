# JavaPA

- Entrada e saída de dados;
- java.lang.System e sua variável pública in;
- Hierarquia de classes de entrada e saída;
- Hierarquia de classe de objetos Java a partir da java.lang.Object;
- Métodos herdados de java.lang.Object;
- Modificadores de acesso: private, protected e public;
- Herança;
- Interfaces;



Para que você possa exercitar estes conteúdos, faça a seguinte tarefa:


- Defina uma interface chamada Ave com os métodos cantar() voar();
* cantar não recebe parâmetro e não retorna valor;
* voar recebe um inteiro e retorna um inteiro; 
- Defina uma classe abstrata chamada Passaro que implementa a interface;
- Defina duas classe concretas chamadas Papagaio e Currupio que estendem Passaro
- Na classe Papagaio:
* O método cantar() mostra na tela a string "Papagaio legal"
* O método voar() retorna o menor valor entre o valor informado e a distancia máxima de voo definida no construtor do objeto  
- Na classe Currupio:
* O método cantar() mostra na tela a string "Piu frio fru fri Pió"
* O método voar() retorna o menor valor entre o valor informado e a distancia máxima de voo definida no construtor do objeto
